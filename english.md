# 英语学习计划（零基础 → 看新闻 + 交流）

## 时间估算

假设每天投入 **1-2 小时**：

- **能看英文新闻**：约 6-12 个月
- **能与外国人交流**：约 1.5-2 年

每天投入 3-4 小时可减半。

## 三阶段学习计划（新概念 1-3 册为主轴）

| 阶段 | 新概念            | 时间        | 目标                                     |
| ---- | ----------------- | ----------- | ---------------------------------------- |
| 一   | 第 1 册（144 课） | 1-3 个月    | 800+ 基础词汇、基本时态、语感建立        |
| 二   | 第 2 册（96 课）  | 3-8 个月    | 1500+ 核心词汇、关键语法、过渡到中等难度 |
| 三   | 第 3 册（60 课）  | 第 8 个月起 | 2000+ 高频词、复杂句式、新闻阅读入门     |

## 每册学习方法

### 第 1 册：语感优先，不做解剖

- 前半本：正常速度过，课文跟读 3 遍（训练发音肌肉记忆）
- 后半本：重点做句型替换练习（key structures），对口语非常有用
- 不逐句翻译，不间断查词，目标是语感不是分析

### 第 2 册：听 → 读 → 复述（核心阶段）

每课三步走：

1. 盲听 2 遍（不看书），能听懂多少算多少
2. 对照课文读一遍，查出影响理解的关键生词（不超过 5 个）
3. 用自己的话口头复述课文内容（哪怕只有 3 句话）

- 96 课不需要全做，做到 **第 60 课左右** 就能明显感到质变
- 课后练习只做选择题，Summary writing 口头完成即可

### 第 3 册：精读 + 真实材料并行

- 继续听→读→复述，复述要求提高：尝试用因果逻辑串联
- 此时同步开始读 BBC / NPR 新闻，新概念 3 句型与新闻高度重合
- 新概念比重逐渐降低，真实材料比重逐渐升高

## 每册精力分配

```
第 1 册: 70% 听+跟读  / 20% 句型替换练习 / 10% 语法查阅
第 2 册: 40% 盲听     / 30% 口头复述      / 30% 阅读
第 3 册: 30% 精读     / 30% 复述          / 40% 真实新闻阅读
```

## 配套资源（补新概念之不足）

| 缺口       | 补充方案                                                                                             |
| ---------- | ---------------------------------------------------------------------------------------------------- |
| 语法框架   | 开局 2-3 周快速过一遍基础语法视频（English Grammar in Use 初级），之后不再系统学，遇到不懂的句子反查 |
| 词汇积累   | Anki / SRS 工具每天 15-20 个高频词，初期用 NGSL 前 1000 词，后期从阅读中提取                         |
| 口语互动   | 新概念没有对话压力，需要 HelloTalk / Tandem 语言交换，或 iTalki 社区导师（$5-10/节），每周至少 3 次  |
| 听力适应性 | 新概念是播音朗读，后期需补充播客（6 Minute English → 正常语速）、YouTube                             |
| 新闻词汇   | 第 3 册偏文学叙事，补充新闻阅读 + Anki 提取新闻高频词                                                |

## 核心原则

1. **可理解输入是唯一引擎**——听不懂/读不懂的材料 = 浪费时间。找 70%-80% 能理解的材料
2. **语法不是前学后用**——遇到不懂的句子时反查，不从头啃语法书
3. **口语的瓶颈不是嘴，是耳朵**——听力到位，口语自然跟上
4. **每天坚持 > 周末突击**——每天 30 分钟远好于周末 4 小时

## 第一个月起步动作

- Week 1-2: 基础语法视频 2 倍速过一遍（只求有印象）
- Week 1-4: Anki 每天 15 个高频词（刷到 500 词）
- Week 2-4: 每天 20 分钟带字幕的简单英文视频
- Week 3-4: 开始读新概念第 1 册，每天 10 页
